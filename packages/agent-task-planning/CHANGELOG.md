## [0.1.0] - 2024-12-29

### Added
- Initial public release
- TodoListPlanner with multi-provider support
- Providers: Anthropic (Claude), OpenAI (GPT-4), Google (Gemini), Ollama (local)
- Guardrails: iteration limits, cost caps, timeouts, content validation
- Structured logging with structlog
- Comprehensive test suite
- Documentation with architecture diagrams
- Example scripts and command-line demo
- Framework-agnostic prompt templates
