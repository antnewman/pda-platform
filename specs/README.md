# PDA Platform Specifications

Technical specifications for the PDA Platform.

## Specifications

| Spec | Version | Description |
|------|---------|-------------|
| [Canonical Model](canonical-model/v1.0/) | 1.0.0 | Universal PM data schema |
| [MCP Servers](mcp-servers/v1.0/) | 1.0.0 | AI agent integration |
| [Benchmarks](benchmarks/v1.0/) | 1.0.0 | PM AI evaluation tasks |
| [Synthetic Data](synthetic-data/v1.0/) | 1.0.0 | Data generation |

## Background

These specifications were developed to support the NISTA Programme and Project Data Standard trial. The PDA Task Force White Paper identified key barriers to AI adoption in project delivery that these specs address.

## Versioning

Specs use semantic versioning. Breaking changes increment the major version.
